# README #

Simple cheap flights search (React/Redux)

![jetui.png](https://bitbucket.org/repo/9a4Xgr/images/916421502-jetui.png)

### Run ###

* `npm install`
* `sudo npm install -g gulp`
* `gulp run`
*  go to the <http://localhost:4000>

### Working parts ###

* Search based on one quote parameter
